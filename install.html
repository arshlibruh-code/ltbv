Give your agents a voice.

One installer wires Claude Code and Codex into a local speech daemon, keeps your existing settings, and gives you one command to verify the whole loop.

3hook events per agent

Install

01 · paste into Terminal
curl -fsSL https://raw.githubusercontent.com/arshlibruh-code/ltbv/main/install.sh | bash
ltbv doctor ltbv open

What happens

02 · no hidden steps
01

Install locally

Clones into ~/.local/share/ltbv, installs locked Python dependencies, and creates ~/.local/bin/ltbv.

02

Wire both agents

Backs up existing settings, preserves unrelated configuration, then adds Stop, UserPromptSubmit, and Notification hooks.

03

Verify the loop

ltbv doctor checks the daemon, TTS engine, both hook sets, local condensing, playback, and browser adapter.

Requirements

03 · checked before mutation
macOSApple Silicon and Intel Macs are accepted.
git + curlGit ships with Xcode Command Line Tools. The installer explains the command if Git is absent.
uvInstalled automatically from Astral when missing.
OllamaOptional. It condenses long replies locally. Without it, ltbv uses deterministic fallback speech.

Daily commands

04 · one control surface

ltbv open

Wakes the daemon quietly and opens the controller at 127.0.0.1:7333.

ltbv doctor

Prints an evidence-backed health report for the complete local loop.

ltbv update

Fast-forwards the installed checkout and reapplies the idempotent hook configuration.

ltbv uninstall

Removes only ltbv hooks, PATH block, command, and installed checkout. Backups remain beside your settings.

Optional browser audio

05 · Arc or Chrome

YouTube ducking

Open arc://extensions or chrome://extensions, enable Developer mode, choose Load unpacked, then select ~/.local/share/ltbv/browser-extension. Enable Browser / YouTube ducking in the controller.

Troubleshooting

06 · shortest path first
command not foundOpen a new terminal. The installer adds ~/.local/bin to ~/.zprofile.
silentRun ltbv doctor, then check SHUT UP and quiet hours in the controller.
Ollama unavailableInstall and start Ollama, or choose none as the condense provider.
update refusedThe installed checkout has local changes. Commit or remove them before running ltbv update.

Safe to rerun: installation is idempotent. Existing Claude and Codex settings are backed up before every hook change.
